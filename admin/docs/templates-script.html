Script Page Templates

Table of Contents
Introduction

The script page templates are the templates that control the appearance of the end-user interfaces such as the account submission form, search results, and comment submission pages. This document will give you an overview of the variables available in each of the script page templates.

Editing Script Page Templates

To edit the script page templates access the Templates > Script Pages function from the control panel menu. This will display a page where you can select which template you want to edit. Once you have selected the template you want to edit, press the Load Template button. You will be returned to the same page, but there will now be a large text box where you can make changes to the template. When you are done making changes, press the Save Template button to save your changes.

Templates and Their Variables

This section will describe all of the available templates and the variables you can use within each template.
    accounts-add.tpl

    This template displays the account submission form

      {$config} - The ToplistX configuration settings; see the Common Variables section for details
      {$errors} - Any error messages that indicate a problem with the information submitted (array value)
      {$account} - The account data that was submitted
      {$categories} - The categories that the user can select from for their account (array value)
      {$user_fields} - The user defined account database fields


    accounts-added.tpl

    This template displays the when an account has been successfully submitted

      {$config} - The ToplistX configuration settings; see the Common Variables section for details
      {$account} - The account data that was submitted
      {$user_fields} - The user defined account database fields


    accounts-banners-links.tpl

    This template displays the when an account holder logs into their account and selects to view banners and links

      {$config} - The ToplistX configuration settings; see the Common Variables section for details
      {$account} - The account data that was submitted
      {$tracking_url} - The URL that the member should link to on their site


    accounts-closed.tpl

    This template displays the when the account submission form is closed (through the General Settings interface)

      No template values


    accounts-edit.tpl

    This template displays the when an account holder logs into their account and selects to edit their account

      {$config} - The ToplistX configuration settings; see the Common Variables section for details
      {$errors} - Any error messages that indicate a problem with the information submitted (array value)
      {$account} - The account information currently on file for the account
      {$categories} - The categories that the user can select from for their account (array value)
      {$user_fields} - The user defined account database fields


    accounts-locked.tpl

    This template displays the when an account holder tries to login and edit their account, but their account is locked

      No template values


    accounts-login.tpl

    This template displays the when an account holder logs into their account and selects to edit their account

      {$config} - The ToplistX configuration settings; see the Common Variables section for details
      {$errors} - Errors encountered during the login process (array value)
      {$login.login_username} - The username entered on the login form
      {$logged_out} - A flag to indicate that the user has logged out of their account


    accounts-overview.tpl

    This template displays the when an account holder logs into their account

      {$config} - The ToplistX configuration settings; see the Common Variables section for details
      {$tracking_url} - The URL that the member should link to on their site
      {$account} - The current account data on file
      {$stats} - The stats from the last 24 hours for the account (array value)


    accounts-password-reset-confirm.tpl

    This template will display when an account holder requests that their password be reset

      {$config} - The ToplistX configuration settings; see the Common Variables section for details
      {$account} - The account data; see the Common Variables section for details


    accounts-password-reset-confirmed.tpl

    This template will be shown when an account holder has confirmed their password reset by e-mail and clicked on the link contained in that e-mail message

      {$config} - The ToplistX configuration settings; see the Common Variables section for details
      {$account} - The account data; see the Common Variables section for details


    accounts-password-reset.tpl

    This template will be shown when an account holder wants to reset their password

      {$config} - The ToplistX configuration settings; see the Common Variables section for details
      {$errors} - If any errors were encountered, this value will contain them (array value)


    default-ranking.tpl

    This template will be used as the template for every new ranking page that you add through the Manage Pages interface. If you would like to make some changes (for example, adding your own custom logo) so that all of your newly added ranking pages automatically have this code, edit this template file. This template uses the functions and variables as described in the Ranking Page Templates section of this manual.

    error-fatal.tpl

    This template will be displayed when a fatal, unrecoverable error has occurred.

      {$config} - The ToplistX configuration settings; see the Common Variables section for details
      {$error} - The error message
      {$trace} - A stack trace showing what functions were called
      {$file} - The file that the error occurred in
      {$line} - The line number where the error occurred


    error-nice.tpl

    This template will be displayed when the user has done some action to trigger an error condition

      {$config} - The ToplistX configuration settings; see the Common Variables section for details
      {$error} - The error message


    global-header.tpl

    This is the global header that will be included at the top of all of the other templates.

      {$config} - The ToplistX configuration settings; see the Common Variables section for details


    global-footer.tpl

    This is the global footer that will be included at the bottom of all of the other templates.

      {$config} - The ToplistX configuration settings; see the Common Variables section for details


    style.css

    This is the style sheet used on all of the script page templates

      No template values
